## Q1：无约束梯度下降

令状态量为
$$X = [x, y]^T$$

梯度为
$$grad f(X) = [x - 3, 10(y - 3)]^T$$

梯度下降迭代公式
$$X_{k+1} = X_k - eta * grad f(X_k)$$

从 `(0, 0)` 出发，停止条件设置为
$$||X - [3, 3]^T|| < 1e-3$$

本机运行结果如下：

| eta | 迭代次数 | 最终 x | 最终 y | 误差 | 状态 |
| --- | ---: | ---: | ---: | ---: | --- |
| 0.01 | 797 | 2.999004 | 3.000000 | 9.963e-04 | 收敛 |
| 0.05 | 157 | 2.999046 | 3.000000 | 9.544e-04 | 收敛 |
| 0.09 | 85 | 2.999010 | 3.000000 | 9.900e-04 | 收敛 |
| 0.10 | 76 | 2.999001 | 3.000000 | 9.990e-04 | 收敛 |
| 0.15 | 50 | 2.999113 | 3.000000 | 8.873e-04 | 收敛 |
| 0.18 | 41 | 2.999122 | 3.000319 | 9.341e-04 | 收敛 |
| 0.181818 | 42 | 2.999344 | 2.999344 | 9.275e-04 | 收敛 |
| 0.19 | 76 | 3.000000 | 2.999001 | 9.990e-04 | 收敛 |
| 0.20 | 100000 | 3.000000 | 0.000000 | 3.000e+00 | 不收敛 |

这个目标函数的Hessian为diag(1, 10)，最大特征值是10，固定学习率梯度下降的稳定范围大致为$0 < eta < 0.2$

当eta接近0.2时，y方向会明显振荡；当eta=0.2时，y方向的误差比例为1 - 10 * eta = -1，误差只会正负翻转，不会变小。所以它看起来一直在动，但其实没有真正靠近目标点。

## Q2

约束为$$x + y \leq 4$$

无约束最优解是$(3, 3)$，但它有$x + y = 6 > 4$，因此最优点应该落在边界$$x + y = 4$$
即
$$g(x, y) = x + y - 4 \leq 0$$

设拉格朗日函数
$$L(x, y, \lambda) = \frac{1}{2}(x - 3)^2 + \frac{10}{2}(y - 3)^2 + \lambda(x + y - 4)$$

KKT 条件为

$$x - 3 + \lambda = 0$$
$$10(y - 3) + \lambda = 0$$
$$\lambda \geq 0$$
$$x + y \leq 4$$
$$\lambda(x + y - 4) = 0$$

约束激活，有$$x + y = 4$$

联立解得

$$\lambda = \frac{20}{11}$$
$$x^* = \frac{13}{11}$$
$$y^* = \frac{31}{11}$$

## Q3：转化为 OSQP 二次规划

OSQP 的标准形式为
$$min \frac{1}{2}X^T P X + q^T X s.t. l \leq A X \leq u$$

把原目标函数展开

```text
f(x, y)
= 1/2(x - 3)^2 + 5(y - 3)^2
= 1/2 x^2 + 5y^2 - 3x - 30y + constant
```

常数项不影响最优解，所以取

$$X = [x, y]^T$$
$$P=
\begin{bmatrix}
     1 & 0 \\
     0 & 10
\end{bmatrix}$$

$$q = \begin{bmatrix}
     -3 \\
     -30
\end{bmatrix}$$

约束写成
$$A = [1  1],
l = [-inf],
u = [4]$$
Python OSQP 输出如下。

```text
Q3 OSQP result
status: solved
x*: [1.1818181818, 2.8181818182]
objective f(x*): 1.8181818182
x + y - 4: -8.589e-12
KKT reference: [1.1818181818, 2.8181818182]
difference: 2.386e-11
```

## 学习记录

一开始我感觉离(3, 3)最近的可行点应该是约束线$x + y = 4$中点附近。但这个问题的目标函数x和y给的权重不一样，优化器更愿意保住y，牺牲相对多的x精度。

Q1 的学习率实验也挺直观，学习率太小会很慢；学习率接近稳定边界时，某些方向会来回振荡。

这次和 AI 的对话主要做了三件事，先从 PDF 里提取题目要求，再一起推导 Q1/Q2/Q3 的数学形式，最后把推导整理成可以运行的代码和 README。
